# 맥에서 MP3 여러 개를 아이폰 오디오북으로 만드는 법 (챕터 포함)

> **"Books 앱에서 챕터까지 딱딱 나뉘는 진짜 오디오북 만들기"**  
> 터미널 초보도 따라할 수 있게, 명령어 하나씩 차근차근.

---

## 왜 M4B인가?

아이폰 Books 앱은 `.m4b` 포맷을 오디오북으로 인식한다.  
MP3 파일 여러 개를 그냥 복사해봤자 "음악"으로 취급되거나 제각각 따로 열린다.  
M4B로 묶으면:

- Books 앱에서 **하나의 오디오북**으로 인식
- **챕터별 이동** 가능 (각 MP3가 챕터로 분리됨)
- 청취 위치 자동 저장

---

## 준비물

- Mac (macOS 어떤 버전이든 OK)
- MP3 파일들 (한 폴더에 모아두기)
- 터미널 (기본 내장 앱)
- 약 15~30분 (설치 포함)

---

## STEP 1. Homebrew 설치

Homebrew는 맥에서 개발 도구를 쉽게 설치할 수 있게 해주는 패키지 매니저다.  
이미 설치되어 있다면 이 단계는 건너뛰어도 된다.

터미널을 열고 아래 명령어를 붙여넣는다:

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

- 비밀번호 입력 요청이 뜨면 맥 로그인 비밀번호를 입력한다 (입력해도 화면에 표시 안 됨, 정상)
- `Press RETURN/ENTER to continue` 메시지가 나오면 **엔터키**를 누른다
- 설치에 5~15분 정도 걸릴 수 있으니 기다린다

---

## STEP 2. FFmpeg 설치

FFmpeg는 오디오/영상 변환의 핵심 도구다.  
Homebrew 설치가 끝나면 아래 명령어를 입력한다:

```bash
brew install ffmpeg
```

설치 완료 후 터미널에 `%` 프롬프트가 다시 나타나면 성공이다.

---

## STEP 3. MP3 파일 정리

변환할 MP3 파일들을 **한 폴더**에 모아둔다.  
파일 이름 앞에 `01_`, `02_`, `03_` 같은 번호를 붙여두면 챕터 순서가 자동으로 맞춰진다.

예시:
```
01_Introduction.mp3
02_Chapter1.mp3
03_Chapter2.mp3
...
```

---

## STEP 4. 터미널에서 폴더로 이동

터미널에서 MP3 파일이 있는 폴더로 이동한다.  
폴더가 데스크탑에 있다면:

```bash
cd ~/Desktop/폴더이름
```

> **예시:** 폴더 이름이 `DragonMaster-19`라면  
> `cd ~/Desktop/DragonMaster-19`

이동 후 파일 목록을 확인한다:

```bash
ls *.mp3
```

파일들이 쭉 출력되면 정상이다.

---

## STEP 5. 파일 목록 생성

MP3 파일 목록을 정렬된 순서로 저장한다:

```bash
ls *.mp3 | sort > filelist.txt && cat filelist.txt
```

터미널에 파일 이름들이 **01, 02, 03...** 순서로 출력되면 성공이다.

---

## STEP 6. FFmpeg용 concat 파일 생성

FFmpeg가 읽을 수 있는 형식으로 목록을 변환한다:

```bash
while IFS= read -r f; do echo "file '$f'"; done < filelist.txt > concat.txt && cat concat.txt
```

아래와 같이 각 파일 앞에 `file '...'` 형식으로 출력되면 성공이다:

```
file '01_Introduction.mp3'
file '02_Chapter1.mp3'
...
```

---

## STEP 7. MP3 전체 합치기 (임시 파일 생성)

모든 MP3를 하나의 M4A 파일로 합친다.  
파일 수에 따라 수 분이 걸릴 수 있다:

```bash
ffmpeg -f concat -safe 0 -i concat.txt -c:a aac -b:a 128k temp_merged.m4a -y
```

완료되면 같은 폴더에 `temp_merged.m4a` 파일이 생성된다.

---

## STEP 8. 챕터 메타데이터 생성

각 MP3의 길이를 측정해서 챕터 정보를 담은 메타데이터 파일을 만든다:

```bash
mapfile -t files < <(ls *.mp3 | sort)

echo ";FFMETADATA1" > chapters.txt
start=0
for f in "${files[@]}"; do
  duration=$(ffprobe -v error -show_entries format=duration \
    -of default=noprint_wrappers=1:nokey=1 "$f")
  duration_ms=$(echo "$duration * 1000 / 1" | bc)
  end=$((start + duration_ms))
  title="${f%.mp3}"
  echo "" >> chapters.txt
  echo "[CHAPTER]" >> chapters.txt
  echo "TIMEBASE=1/1000" >> chapters.txt
  echo "START=$start" >> chapters.txt
  echo "END=$end" >> chapters.txt
  echo "title=$title" >> chapters.txt
  start=$end
done
```

완료되면 `chapters.txt` 파일이 생성된다.

---

## STEP 9. 최종 M4B 파일 생성

합친 오디오와 챕터 정보를 결합해서 최종 오디오북 파일을 만든다:

```bash
ffmpeg -i temp_merged.m4a -i chapters.txt \
  -map_metadata 1 -codec copy output.m4b -y
```

완료되면 `output.m4b` 파일이 생성된다. 🎉

---

## STEP 10. 임시 파일 정리

작업 중 생성된 임시 파일들을 삭제한다:

```bash
rm -f temp_merged.m4a filelist.txt concat.txt chapters.txt
```

---

## STEP 11. 아이폰으로 전송

1. USB 케이블로 아이폰과 맥을 연결한다
2. Finder를 열면 사이드바에 아이폰이 나타난다
3. 아이폰을 클릭 → **Books** 탭 선택
4. `output.m4b` 파일을 Books 영역으로 **드래그 앤 드롭**
5. 아이폰 Books 앱에서 확인!

---

📚 output

## 전체 명령어 요약 (복붙용)

```bash
# 1. 폴더로 이동
cd ~/Desktop/폴더이름

# 2. 파일 목록 생성
ls *.mp3 | sort > filelist.txt

# 3. concat 파일 생성
while IFS= read -r f; do echo "file '$f'"; done < filelist.txt > concat.txt

# 4. MP3 합치기
ffmpeg -f concat -safe 0 -i concat.txt -c:a aac -b:a 128k temp_merged.m4a -y

# 5. 챕터 메타데이터 생성
mapfile -t files < <(ls *.mp3 | sort)
echo ";FFMETADATA1" > chapters.txt
start=0
for f in "${files[@]}"; do
  duration=$(ffprobe -v error -show_entries format=duration \
    -of default=noprint_wrappers=1:nokey=1 "$f")
  duration_ms=$(echo "$duration * 1000 / 1" | bc)
  end=$((start + duration_ms))
  title="${f%.mp3}"
  echo "" >> chapters.txt
  echo "[CHAPTER]" >> chapters.txt
  echo "TIMEBASE=1/1000" >> chapters.txt
  echo "START=$start" >> chapters.txt
  echo "END=$end" >> chapters.txt
  echo "title=$title" >> chapters.txt
  start=$end
done

# 6. 최종 M4B 생성
ffmpeg -i temp_merged.m4a -i chapters.txt \
  -map_metadata 1 -codec copy output.m4b -y

# 7. 임시 파일 정리
rm -f temp_merged.m4a filelist.txt concat.txt chapters.txt

echo "✅ 완료! output.m4b 파일을 아이폰 Books 앱으로 전송하세요."
```

## 자주 묻는 질문

**Q. 파일명에 한글이 있어도 되나요?**  
A. 됩니다. FFmpeg는 한글 파일명도 처리합니다.

**Q. MP3가 아닌 다른 포맷(AAC, WAV 등)도 되나요?**  
A. 됩니다. `ls *.mp3` 부분을 `ls *.wav` 등으로 바꾸면 됩니다.

**Q. 챕터 이름을 바꾸고 싶어요.**  
A. MP3 파일명이 곧 챕터명이 됩니다. 파일명을 원하는 챕터명으로 바꿔두면 됩니다.

**Q. output.m4b 파일명을 바꾸고 싶어요.**  
A. 명령어에서 `output.m4b`를 원하는 이름으로 바꾸면 됩니다.  
예: `DragonMaster19.m4b`
